HTML é a linguagem usada para criar páginas na web. Ela significa "Hypertext Markup Language" ou "Linguagem de Marcação de Hipertexto".
Basicamente, HTML é como uma espécie de esqueleto que dá a estrutura para as informações que serão exibidas em uma página na web.

Ele usa "tags" (ou etiquetas) para definir os diferentes elementos na página, como títulos, parágrafos, imagens, links e muitos outros.
Além disso, HTML também pode ser usado para definir outros aspectos da página, como a cor do fundo, o tamanho da fonte e a posição dos
elementos na tela.

Em resumo, HTML é a linguagem usada para criar o "esqueleto" das páginas da web. Ela usa tags para definir diferentes elementos na página
e seus atributos, como cores e tamanhos. Sem HTML, não teríamos as páginas da web que usamos todos os dias!

    Principais tags HTML:Principais tags HTML:
  • Body: contém todo o conteúdo da página, incluindo texto, imagens, vídeos e outros elementos.
  • h1, h2, h3 etc.: usadas para definir títulos em diferentes níveis de importância.
  • p:define um parágrafo de texto.
  • nav: utilizada para definir uma secção de navegação
  • a:cria um link para outra página da web ou para uma seção da mesma página.
  • ul e li: usados para criar listas com marcadores.
  • table, tr e td:usados para criar tabelas com linhas e colunas.
  • form e inputs:usados para criar formulários para coletar informações do usuário.
  • main: utilizado para agrupar o conteúdo principal da página
  • footer: utilizado para criar secção de rodapé de uma página

O objetivo destá página é mostra como deve ser estruturada uma página HTML, utilizando devidamente suas principais tags.
